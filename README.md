# Video Playlist Scraper

一个 Claude Code Skill，用于从视频平台抓取片单数据并输出 Excel 表格。

## 支持平台

| 平台 | 状态 |
|------|------|
| 爱奇艺 | 已验证 |
| 腾讯视频 | 待探索（首次使用时自动分析） |
| 优酷 | 待探索 |
| 芒果TV | 待探索 |
| B站 | 待探索 |

> Skill 采用引导式设计，不预置脚本。首次抓取新平台时，Claude 会实时分析网站结构并动态生成抓取脚本，因此理论上支持任意视频平台。

## 工作原理

```
用户: "帮我抓取爱奇艺的独播电视剧"
                │
                ▼
        ┌───────────────┐
        │  1. 需求确认    │  确认平台、类型、筛选条件、输出格式
        └───────┬───────┘
                ▼
        ┌───────────────┐
        │  2. 平台探索    │  分析目标网站的页面结构和加载机制
        └───────┬───────┘
                ▼
        ┌───────────────┐
        │  3. 脚本生成    │  动态生成 Python + Playwright 抓取脚本
        └───────┬───────┘
                ▼
        ┌───────────────┐
        │  4. 执行抓取    │  运行脚本，实时显示进度
        └───────┬───────┘
                ▼
        ┌───────────────┐
        │  5. 输出验证    │  生成 Excel，验证数据完整性
        └───────────────┘
                │
                ▼
          片单.xlsx 📊
```

## 核心特性

- **引导式工作流**：不依赖预置脚本，实时分析页面 DOM 结构，适应网站更新
- **反爬绕过**：使用 Playwright 浏览器自动化，天然支持 JS 渲染页面
- **经验积累**：成功抓取的平台结构信息会记录到 `references/platform-patterns.md`，后续复用
- **灵活筛选**：支持按标签筛选（独播/自制/VIP/热播等）
- **Excel 输出**：按类型分 sheet，格式规范

## 安装

### 环境要求

- Python 3
- Claude Code

### 安装依赖

```bash
pip install playwright pandas openpyxl
python -m playwright install chromium
```

### 安装 Skill

将本仓库克隆到 Claude Code 的 skills 目录：

```bash
git clone git@github.com:gmyifan/-skill.git ~/.claude/skills/video-playlist-scraper
```

安装后 Claude Code 会自动识别该 Skill。

## 使用方式

在 Claude Code 中用自然语言触发即可：

```
帮我抓取爱奇艺的独播片单
```

```
整理腾讯视频的电视剧和电影列表，输出 Excel
```

```
抓取优酷的独播综艺节目
```

Claude 会按照 5 个阶段引导你完成抓取。

## 输出示例

以爱奇艺独播抓取为例，输出 `爱奇艺独播片单.xlsx`：

| Sheet | 内容 | 数量（参考） |
|-------|------|-------------|
| 电视剧 | 独播电视剧列表 | 317 条 |
| 电影 | 独播电影列表 | 84 条 |
| 综艺 | 独播综艺列表 | 252 条 |
| 动漫 | 独播动漫列表 | 89 条 |

每条记录包含：

| 片名 | 类型 | 网址链接 |
|------|------|---------|
| 你好1983 | 电视剧 | https://www.iqiyi.com/v_xxx.html |

## 注意事项

- 各平台列表页通常有数量上限（如爱奇艺每类最多 768 条），这是平台本身的限制
- 视频平台页面结构可能随时更新，如果抓取失败，Skill 会重新探索页面结构
- 部分内容可能需要登录才能访问（如 VIP 专区）
- 请合理使用，遵守目标网站的服务条款
